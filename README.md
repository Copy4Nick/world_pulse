# World Pulse

Интерактивная 3D-карта активных мировых кризисов в реальном времени. Приложение агрегирует свежие новости, анализирует их с помощью LLM и отображает 15 наиболее значимых ситуаций на вращающемся глобусе с корректной сменой дня и ночи.

![World Pulse](./src/assets/hero.png)

---

## Возможности

- **3D-глобус** с реалистичной текстурой Земли (день/ночь) и границами стран — рендеринг через кастомный GLSL-шейдер
- **15 активных ситуаций** из реальных новостей: конфликты, протесты, природные катастрофы, экономические и политические кризисы
- **ИИ-аналитик**: Gemini 2.0 Flash (через OpenRouter) обогащает новостные сигналы экспертным контекстом и выдаёт структурированные брифинги
- **Фильтрация** по типу события прямо в сайдбаре
- **Детальные карточки**: краткий анализ, длительность ситуации, конкретные последствия
- **Автообновление** каждые 10 минут с кешем на бэкенде
- **Live-часы UTC** в заголовке

## Быстрый старт

### 1. Клонируйте репозиторий

```bash
git clone <url>
cd world-pulse
```

### 2. Установите зависимости

```bash
npm install
```

### 3. Настройте переменные окружения

Скопируйте шаблон и заполните ключи:

```bash
cp .env.example .env
```

Откройте `.env` и укажите свои ключи:

```env
OPENROUTER_API_KEY=sk-or-...   # https://openrouter.ai/keys
NEWS_API_KEY=...               # https://newsapi.org/account
```

> **Важно:** никогда не коммитьте `.env` с реальными ключами. Убедитесь, что `.env` добавлен в `.gitignore`.

### 4. Запустите бэкенд

```bash
npm run server
# → http://localhost:3001
```

### 5. Запустите фронтенд (в отдельном терминале)

```bash
npm run dev
# → http://localhost:5173
```

## Получение API-ключей

**NewsAPI** — бесплатный план (100 запросов/сутки):
1. Зарегистрируйтесь на [newsapi.org](https://newsapi.org)
2. Скопируйте ключ из личного кабинета

**OpenRouter** — доступ к Gemini и другим моделям:
1. Зарегистрируйтесь на [openrouter.ai](https://openrouter.ai)
2. Создайте ключ в разделе Keys
3. Пополните баланс (Gemini 2.0 Flash очень дёшев)
